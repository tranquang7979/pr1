1. start NodeJS Command Prompt.
2. point to app folder "cd [yourpath]\awesome-chat\awesome-chat"
3. run command "node app.js"
4. open browser and go to "http://localhost:3000/landing"
5. type username and login.
6. create room or join existed rooms.
7. start chatting.

note: you can open other browser and repeat from step 4 -> 7 to test
